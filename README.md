# bulkviz

bulkviz is a publication-first R package for elegant bulk transcriptomics visualizations.

Current development focus:
- publication-quality PCA plots
- volcano plots
- heatmaps
- gene expression plots
- unified themes and palettes

## Design goals
- beautiful defaults
- low-friction APIs
- consistent figure grammar
- bulk transcriptomics first
- visualization only, not upstream DE/enrichment analysis

## Planned core functions
- `pca_plot()`
- `volcano_plot()`
- `heatmap_plot()`
- `gene_plot()`
- `sample_heatmap()`
- `theme_pub()`
- `palette_pub()`

## Development status
Early scaffold.
